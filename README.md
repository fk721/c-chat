# c-chat [WIP]
Terminal chat written in C, styled with ncurses. 

### Build
```bash
mkdir build
cd build
cmake ..
make
```

### Running
Start the server:
```bash
./build/bin/c-server
```

Start a client(s):
```bash
./build/bin/c-server USERNAME
```
